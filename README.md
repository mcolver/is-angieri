# is-angieri

## A professional portfolio.


### Runbooks

#### Installation
>`npm install`
>`brew install mongo`

**Run mongod without sudo:**

*Create folder*
>`sudo mkdir -p /data/db/`

*Give yourself permission to the folder*
>```
sudo chown `id -u` /data/db
``

#### Run application
1st shell tab:
>`mongod`

2nd shell tab
>`mongo`

3rd shell tab
>`npm start`

Navigate to 'localhost:3000/'

To test:
>`npm test`
